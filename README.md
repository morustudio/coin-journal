# 코인 매매일지

바이낸스 선물 매매 전용 일지. 로컬 저장 + 구글 시트 동기화 지원.

## 기능

- **매매 기록**: 진입가, 청산가, 레버리지, 수수료 입력 → 손익/손익률 자동계산
- **통계**: 오늘 손익, 누적 손익, 승률 (LONG/SHORT 분리)
- **차트**: 기간별 누적 손익 그래프
- **CSV**: 내보내기 / 가져오기
- **구글 시트 동기화**: Apps Script 연동으로 클라우드 백업
- **다크/라이트 모드**: 상단 버튼으로 전환
- **모바일 지원**: 반응형 레이아웃, 모달 Bottom Sheet

## 구글 시트 연동 설정 (선택사항)

### 1단계 — 구글 시트 생성

1. [Google Drive](https://drive.google.com) 접속
2. 새 스프레드시트 생성
3. 파일명은 자유롭게 설정

### 2단계 — Apps Script 설정

1. 스프레드시트 상단 메뉴 → **확장 프로그램** → **Apps Script**
2. 기존 코드 전체 삭제
3. 앱 내 **구글 시트** 버튼 → 설정 모달의 코드를 전체 복사해서 붙여넣기
4. **저장** (Ctrl+S)

### 3단계 — 웹 앱으로 배포

1. 오른쪽 상단 **배포** → **새 배포**
2. 배포 유형: **웹 앱** 선택
3. 설정:
   - 설명: `코인 매매일지` (자유)
   - 실행 계정: **나 (본인 계정)**
   - 액세스 권한: **모든 사용자** ← 반드시 이렇게 설정
4. **배포** 클릭 → 권한 허용
5. 생성된 **웹 앱 URL** 복사

> ⚠️ 액세스 권한을 "모든 사용자"로 설정해야 앱에서 접근 가능합니다.  
> URL을 알아야만 접근할 수 있으므로 보안상 문제없습니다.

### 4단계 — 앱에서 연결

1. 매매일지 상단 **구글 시트** 버튼 클릭
2. 복사한 URL 붙여넣기
3. **연결 저장** 클릭 → 연결 테스트 자동 진행
4. 성공 메시지 확인 후 완료

### 이후 사용

- 매매 기록 추가/수정/삭제 시 상단 상태 표시가 **주황(●)** 으로 변경
- 테이블 상단의 **시트 동기화** 버튼 클릭 → 구글 시트에 저장
- 상태 표시 **초록(✓)** = 동기화 완료

## 파일 구조

```
index.html   — 화면 구조 (마크업)
style.css    — 스타일 (다크/라이트 모드, 반응형)
script.js    — 로직 (저장, 계산, 차트, 동기화)
```
